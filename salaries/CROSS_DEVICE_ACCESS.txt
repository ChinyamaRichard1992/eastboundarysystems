🔥 WINDOWS FIREWALL FIX (If Still Not Working)

If you can't access from other devices:

OPTION 1: Quick Test (Disable Firewall Temporarily)
────────────────────────────────────────────────────
1. Open Windows Settings
2. Go to: Windows Security → Firewall & Network
3. Click your network (Private/Public)
4. Turn OFF "Microsoft Defender Firewall"
5. Try accessing from phone again
6. If it works, firewall was blocking it
7. Turn firewall back ON
8. Follow Option 2 to allow Apache permanently

OPTION 2: Allow Apache Through Firewall (Permanent Fix)
────────────────────────────────────────────────────
1. Press Windows Key + R
2. Type: wf.msc
3. Press Enter (Windows Firewall opens)
4. Click "Inbound Rules" on left
5. Click "New Rule" on right
6. Select "Port" → Next
7. Select "TCP" and enter: 80
8. Click Next
9. Select "Allow the connection"
10. Click Next
11. Check all: Domain, Private, Public
12. Click Next
13. Name: "XAMPP Apache"
14. Click Finish
15. ✅ Done! Other devices can now connect!

OPTION 3: XAMPP Built-in Firewall Fix
────────────────────────────────────────────────────
1. Open XAMPP Control Panel
2. Click "Config" next to Apache
3. Select "httpd.conf"
4. Find: Listen 80
5. Should say: Listen 80 (not Listen 127.0.0.1:80)
6. Save file
7. Restart Apache in XAMPP
8. ✅ Should work now!

⚡ TROUBLESHOOTING

Problem: "Site can't be reached" on phone
Solution:
1. Check both devices on same WiFi
2. Check Apache is running (XAMPP Control Panel)
3. Check Windows Firewall (see fix above)
4. Try: http://192.168.215.96/ (XAMPP dashboard)

Problem: "Connection refused"
Solution:
1. Restart Apache in XAMPP
2. Check firewall settings
3. Make sure IP is correct: 192.168.215.96

Problem: Verification email link doesn't work
Solution:
1. Email now has correct IP
2. But if old email, use manual method
3. Go to verification page manually
4. Enter code from email

Problem: IP address changed
Solution:
1. Your IP might change if you restart router
2. Check current IP: Open Command Prompt, type: ipconfig
3. Look for "IPv4 Address" under your WiFi adapter
4. Use that new IP address
