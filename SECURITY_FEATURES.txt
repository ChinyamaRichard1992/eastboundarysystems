🔄 HOW TO USE THE SYSTEM
─────────────────────────────────────────────────────────────────────────────

FIRST TIME USER:
1. Navigate to: http://localhost/eastboundarysystems/
2. Click "Sign Up" tab
3. Fill in: Name, Email, Password, Confirm Password, Role
4. Ensure password meets all requirements (checkmarks turn green)
5. Agree to Terms & Conditions
6. Click "Create Account"
7. System redirects to Login automatically
8. Enter email and password
9. Click "Login" to access the system

RETURNING USER:
1. Navigate to: http://localhost/eastboundarysystems/
2. If session valid → Auto-redirected to dashboard
3. If session expired → Enter email and password
4. Optional: Check "Remember Me" to save email
5. Click "Login"
